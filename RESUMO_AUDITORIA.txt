🏗️ ANÁLISE ARQUITETURAL
─────────────────────────────────────────────────────────────────────────────────

SOLID VIOLATIONS:

❌ S (Single Responsibility):
   compute() faz: clique + produção + multipliers
   → Deve ser 3 funções separadas

❌ O (Open/Closed):
   Adicionar novo evento = modificar EVENTS array
   → Deve ter factory pattern

❌ I (Interface Segregation):
   fireClick() faz: lógica + UI + áudio tudo junto
   → Deve separar em functions pequenas

❌ D (Dependency Inversion):
   Acoplado ao DOM e localStorage
   → Deve abstrair camada de persistência

🚨 GARGALOS CRÍTICOS (RANKED)
─────────────────────────────────────────────────────────────────────────────────

#1 🔴 CRÍTICO: DOM Event Listeners não removidos
    Impacto: Memory leak
    Causa: addEventListener sem cleanup
    Fix: AbortController com scope
    Effort: ⚡ 30 min
    Código:
      const controller = new AbortController();
      btn.addEventListener('click', handler, { signal: controller.signal });
      // Cleanup: controller.abort();

#2 🔴 CRÍTICO: buildBldTab() recria 150+ elementos
    Impacto: 15-25ms bloqueante a cada 1.5s
    Causa: innerHTML = '' + createElement loop
    Fix: Dirty flag + partial DOM update
    Effort: ⚡ 1 hora
    Código:
      let isDirty = { bld: false };
      if (isDirty.bld) buildBldTab();
      isDirty.bld = false;

#3 🟡 MÉDIO: Hair array sem limite superior
    Impacto: 40ms lag spike se 50+ hairs
    Causa: hairs.push() sem cap
    Fix: Max 40 hairs, FIFO com pop
    Effort: ⚡ 30 min
    Código:
      const MAX_HAIRS = 40;
      if (hairs.length >= MAX_HAIRS) hairs.shift();

#4 🟡 MÉDIO: JSON.parse silent failure
    Impacto: Perda invisível de save
    Causa: catch(e) { } sem logging
    Fix: console.error + recovery
    Effort: ⚡ 15 min
    Código:
      catch(e) {
        console.error('[LOAD] Corrupted:', e.message);
        G = newState();
      }

#5 🟡 MÉDIO: Combo decay sem debounce
    Impacto: 0.5ms overhead por frame
    Causa: Recalc em todo frame
    Fix: Update interval de 100ms
    Effort: ⚡ 20 min
